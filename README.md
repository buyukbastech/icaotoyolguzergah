# ICA Otoyol Güzergah & Ücret Hesaplayıcı

ICA İçtaş Altyapı İstanbul Kuzey Marmara Otoyolu güzergah ve ücret hesaplama uygulaması. 

## Özellikler

- **Resmî KM Verileri**: PDF harita verilerine dayalı sıfır sapmalı mesafe hesaplama.
- **Ücret Hesaplayıcı**: 2026 resmî tarifesine uygun Gişe ve Serbest Geçiş (SGS) ücretleri.
- **Karbon Ayak İzi**: Yolculuk bazlı CO2 salınımı tahmini.
- **İnteraktif Harita**: Tüm gişe ve geçiş noktalarının görselleştirilmesi.
- **Dinamik Grafikler**: Güzergah boyu ücret projeksiyonu ve araç sınıfı karşılaştırmaları.

## Kurulum

Projeyi yerelinizde çalıştırmak için:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

## Teknolojiler

- React & TypeScript
- Vite
- Tailwind CSS
- Framer Motion (Animasyonlar)
- Recharts (Grafikler)
- Lucide React (İkonlar)
